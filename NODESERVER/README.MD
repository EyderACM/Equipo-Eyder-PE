# Backend of Frugalist 
To run this project type the next commands:
* $npm install
* $npm run dev

If you need ident the code use the command:
* $npm run prettier

**Dependencies:**
* body-parser 
* cross-env express 
* nodemon 
* babel-cli 
* babel-plugin-transform-object-rest-spread 
* babel-preset-env
* eslint
* eslint-config-equimper
* prettier
* eslint-config-prettier
