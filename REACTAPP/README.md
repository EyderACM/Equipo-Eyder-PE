# Application of Frugalist
To run this proyect type the next commands (FirstTime): 
* $npm install expo-cli --global 
* $npm install
* $npm start 

If you need ident the code use the command:
* $npm run prettier (Not configured)

**Dependencies:**
* expo-cli
